# Iteración 2 – Ventas, Tickets e Inventario

Esta iteración se centra en extender el sistema **AppJB.Ciber** para integrar:

- Registro de ventas de productos (snacks, bebidas, etc.).
- Cálculo de totales (subtotal, IVA, descuentos/promociones).
- Generación de ticket impreso o en PDF.
- Actualización automática del inventario.

La iteración está pensada como un incremento funcional sobre la Iteración 1 (Cliente + Sesiones), reutilizando actores y parte del dominio ya definido.
